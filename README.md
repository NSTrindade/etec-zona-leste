# ETEC Zona Leste - Website Institucional

[Clique aqui para ir ao VÍDEO](https://youtu.be/9AQJfdeXCTo)


Um website moderno e responsivo para uma instituição de ensino técnico, desenvolvido com **Laravel 11**, **Blade Templates** e **Tailwind CSS**.

## 📋 Características

- **Página Inicial (Home)**: Banner de destaque, cursos em destaque e notícias recentes (Chapa, IR, Feira Tecnológica).
- **Página Sobre**: História da instituição, missão, visão, valores e infraestrutura.
- **Página Cursos**: Lista completa de cursos técnicos com descrições e competências.
- **Página Contato**: Formulário de contato, informações de departamentos, FAQ e **Mapa Interativo (Google Maps)**.
- **Design Responsivo**: Interface moderna adaptada para desktop, tablet e mobile.
- **Dados Mockados**: Todos os dados simulados diretamente no Controller (sem necessidade de banco de dados inicial).
- **Tailwind CSS**: Estilização via CDN para configuração rápida e performance.

## 🛠️ Tecnologias Utilizadas

- **Backend**: Laravel 11
- **Frontend**: Blade Templates + Tailwind CSS (via CDN)
- **Ícones**: Font Awesome 6.4.0
- **Fontes**: Google Fonts (Poppins)
- **Mapas**: Google Maps Embed API
- **Servidor**: PHP Built-in Server
